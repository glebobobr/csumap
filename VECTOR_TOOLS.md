# Векторные инструменты редактора

## Новые возможности

### 1. Векторные примитивы с расширенными параметрами

#### Прямоугольник со скруглением углов
- **Функция**: `makeRoundedRectangle(center, widthM, heightM, radiusM, angleDeg)`
- **Параметры**:
  - `center` — центр [lng, lat]
  - `widthM` — ширина в метрах
  - `heightM` — высота в метрах
  - `radiusM` — радиус скругления углов в метрах
  - `angleDeg` — угол поворота в градусах
- **Использование**: Кнопка "Скругл." в панели фигур

#### Кольцо (круг с выдавливанием)
- **Функция**: `makeRing(center, outerRadiusM, innerRadiusM, steps)`
- **Параметры**:
  - `center` — центр [lng, lat]
  - `outerRadiusM` — внешний радиус в метрах
  - `innerRadiusM` — внутренний радиус в метрах (0 = обычный круг)
  - `steps` — количество точек (по умолчанию 48)
- **Использование**: Кнопка "Кольцо" в панели фигур

#### Линия с толщиной (выдавливание радиуса)
- **Функция**: `makeThickLine(points, thicknessM, capStyle)`
- **Параметры**:
  - `points` — массив точек [[lng, lat], ...]
  - `thicknessM` — толщина линии в метрах
  - `capStyle` — стиль концов: `'butt'`, `'round'`, `'square'`
- **Описание**: Преобразует линию (LineString) в полигон с заданной толщиной
- **Применение**: Для создания дорог, тропинок, коридоров

### 2. Булевы операции

Все булевы операции работают только с полигонами (Polygon/MultiPolygon). Для выполнения операции:
1. Выберите 2 или более объекта (Ctrl+Click или рамкой)
2. Нажмите кнопку нужной операции

#### Объединение (Union)
- **Функция**: `union(features)`
- **Описание**: Объединяет все выбранные полигоны в один, удаляя внутренние пересечения
- **Пример**: Объединить несколько зданий в один комплекс

#### Вычитание (Difference)
- **Функция**: `difference(features)`
- **Описание**: Вычитает все последующие объекты из первого
- **Пример**: Вырезать окна/двери из здания, создать внутренний двор

#### Пересечение (Intersection)
- **Функция**: `intersection(features)`
- **Описание**: Оставляет только общую область всех выбранных объектов
- **Пример**: Найти зону пересечения двух территорий

#### Исключающее ИЛИ (XOR)
- **Функция**: `xor(features)`
- **Описание**: Оставляет области, где объекты НЕ пересекаются
- **Пример**: Создать сложные формы с вырезами

## Архитектура

### Модули

#### `src/editor/tools/primitives.js`
Базовые векторные примитивы:
- `makeRoundedRectangle()` — прямоугольник со скруглением
- `makeThickLine()` — линия с толщиной
- `makeRing()` — кольцо/круг с выдавливанием

#### `src/editor/tools/boolean-ops.js`
Булевы операции над полигонами:
- `union()` — объединение
- `difference()` — вычитание
- `intersection()` — пересечение
- `xor()` — исключающее ИЛИ
- `areAllPolygons()` — проверка типов геометрии

Использует библиотеку `polygon-clipping` для выполнения операций.

#### `src/editor/tools/shapes.js`
Экспортирует все примитивы и существующие функции генерации фигур.

### Интеграция

Все новые инструменты интегрированы в:
- **Toolbar** (`toolbar.js`) — кнопки для фигур и булевых операций
- **ShapeModal** (`shape-modal.js`) — параметры для новых фигур
- **Editor** (`editor.js`) — обработка событий и выполнение операций

## Использование

### Создание фигуры с параметрами
1. Нажмите кнопку нужной фигуры (Круг, Кольцо, Скругл. и т.д.)
2. Настройте параметры в модальном окне
3. Нажмите "Вставить на карту"
4. Заполните свойства объекта (название, тип и т.д.)

### Выполнение булевой операции
1. Выберите 2+ полигона (удерживайте Ctrl для множественного выбора)
2. Нажмите кнопку нужной операции (Объединить, Вычесть и т.д.)
3. Исходные объекты будут удалены, результат автоматически выбран

### Программное использование

```javascript
import { makeRoundedRectangle, makeThickLine, makeRing } from './tools/primitives.js'
import { union, difference, intersection, xor } from './tools/boolean-ops.js'

// Создать скруглённый прямоугольник
const rect = makeRoundedRectangle(
  [61.318987, 55.177196], // центр
  60,  // ширина 60м
  30,  // высота 30м
  5,   // радиус скругления 5м
  0    // без поворота
)

// Создать линию с толщиной
const road = makeThickLine(
  [[61.318, 55.177], [61.319, 55.178], [61.320, 55.179]], // точки
  3,       // толщина 3м
  'round'  // скруглённые концы
)

// Объединить два полигона
const feature1 = { type: 'Feature', geometry: rect, properties: {} }
const feature2 = { type: 'Feature', geometry: road, properties: {} }
const merged = union([feature1, feature2])
```

## Технические детали

### Координаты
Все функции работают с географическими координатами [longitude, latitude] и автоматически конвертируют метры в градусы с учётом широты (~55°).

### Константы преобразования
```javascript
M_PER_DEG_LAT = 111320  // метров на градус широты
M_PER_DEG_LNG = 63900   // метров на градус долготы (на широте 55°)
```

### Библиотека polygon-clipping
Используется для выполнения булевых операций. Поддерживает:
- Polygon и MultiPolygon
- Полигоны с дырками (holes)
- Высокая точность вычислений

## Примеры использования

### Создание здания с внутренним двором
1. Создайте внешний прямоугольник (здание)
2. Создайте внутренний прямоугольник (двор)
3. Выберите оба объекта
4. Нажмите "Вычесть" — внутренний прямоугольник вырежется из внешнего

### Создание дороги
1. Нарисуйте линию (инструмент Line)
2. Используйте `makeThickLine()` для преобразования в полигон с толщиной
3. Настройте стиль концов (`round`, `square`, `butt`)

### Объединение зданий в комплекс
1. Создайте несколько прямоугольников (здания)
2. Выберите все здания
3. Нажмите "Объединить" — получится один объект без внутренних границ
